# EP04 Agent Instructions

本資料夾是《AI Agent 基本功》EP04「連接外部工具」專案。

## 專案邊界

- 主要交付物：`slides.html`、`EP04_大綱.md`、`README.md`、`基本功_外部工具連接方式總表.md`、`handoff.md`。
- 本集定位是「外部工具連接方式的分類地圖」，不是 Padlet 實作課程本身。
- 若要修改簡報或大綱，先確認頁數、README、handoff 是否需要同步。

## 可引用的外部 Padlet 模組

- 外部來源：`G:\我的雲端硬碟\2026邀約\Padlet模組`
- 用途：提供 Padlet 課程牆模板、建板規格、檢核清單，供 EP04 或後續 Agent 基本功研習引用。
- 優先閱讀順序：
  1. `00_Padlet模組索引.md`
  2. `08_Padlet建板規格表.md`
  3. 依研習主題選擇 `01_` 到 `06_` 的模板
  4. `09_建板檢核清單.md`
- 引用時要在相關文件註明：「外部引用來源：`G:\我的雲端硬碟\2026邀約\Padlet模組`」。

## Padlet 引用規則

- 可以引用模板架構、區段名稱、建板規格與檢核方式。
- 不要把 Padlet token、登入資訊、實際看板管理權限寫進 repo。
- 實際 Padlet URL、QR Code、活動日期、學校名稱，應寫在當次研習專案或 handoff，不直接覆蓋共用模板。
- 若修改共用模板，必須同步更新 `00_Padlet模組索引.md` 的狀態或說明。
