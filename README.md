## 🗺️ 這集在講什麼

**核心：連接外部工具，先分兩刀。**

1. **是不是 Google 家的服務？**（Gmail／行事曆／Drive／Classroom／NotebookLM／Firebase）
2. **要「開箱即用」還是「完整控制」？**

|  | 非 Google | Google 家 |
|---|---|---|
| **簡單路** | 內建連接器 ／ MCP ＋ API Key | 原生連接器（開關即用） |
| **完整控制路** | MCP ＋ OAuth ／ 本地直存 | 自建 GCP ＋ OAuth 2.0 |

**底層兩條軸**：通道（連接器／MCP／CLI／操控畫面）＝「怎麼接」；授權（免鑰匙／API Key／OAuth）＝「給多大權」。比喻：**通道是水管，授權是水裡流的水。**
